# 09 - 메서드

### 작업1) 수업 데이터를 처리하는 코드를 메서드로 분리하라.
    - 수업 등록 코드를 addVideo() 메서드로 분리한다.
    - 수업 목록 출력 코드를 listVideo() 메서드로 분리한다.
    - 두 메서드가 공통으로 사용하는 변수를 `클래스 변수`로 전환한다.

### 작업2) 회원 데이터를 처리하는 코드를 메서드로 분리하라.
